MMAF to Decouple the Variablity in the Practice Context

This interactive dashboard is to demo and test the MMAF algorithm discussed in the CIGRE-660 paper "Decoupling the variability using Multi-step Moving Average Filters", submitted to 2023 CIGRE Canada Conference (Vancouver, BC, Sept.25 – 28, 2023).

This dashboard takes the wind and solar generation time-sequence (minute-by-minute ) data used in the paper as default data input.

Its input data can be switched to the load time-sequence data used in the paper as input by used the 'load.csv' as the valve  of 'csv' URL parameter.  https://huming66.github.io/mmaf/?csv=load.csv

Besides, this dashboard can take any time-sequence (with certain fixed time interval) data prepared in CSV format, here is the default CSV data file used in this dashboard.

To use the your own prepared CSV data in this dashboard, you just drag and drop your prepared CSV file with the time-sequencial data into the browser page.

Your data will be used ONLY inside your broswer, there is no code in the page to collect or send any of your data outside your broswer.
